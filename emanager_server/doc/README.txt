Please edit serverConfig.json properly before moving forward.
Please find instalation instructions in doc/instructions.txt.
Launch the server from a terminal by typing:

./launch_emanager_server.sh

if you want to redirect out use stdbuf in the following way:

stdbuf -oL ./launch_emanager_server.sh &> log.txt

Locking screen should be disabled in the participating devices.

You might need to build the Android app first (Normapp). 
Please check the Android app project build folder.
